# Anthropic Agentic AI Engineering

A structured course on engineering modern AI-powered software systems using **agentic architectures** and **enterprise AI workflows**. The material is inspired by industry concepts and tooling associated with Anthropic, Claude, MCP, and contemporary enterprise agentic AI platforms.

## Project Overview

This repository contains an 8-module curriculum that takes learners from foundational concepts to production-grade agentic systems. Across the modules you will learn how to:

- Build autonomous AI agents
- Design multi-agent systems
- Integrate AI tools using MCP (Model Context Protocol)
- Develop RAG-based knowledge systems
- Engineer AI-assisted software workflows
- Deploy and orchestrate production AI systems
- Apply governance, security, and responsible AI practices

## Course Architecture

The curriculum is organized as a progressive 8-module learning path. Each module builds on the previous, moving from foundational concepts through to production deployment and governance.

```
Foundations ─▶ Prompting ─▶ MCP ─▶ Single Agents ─▶ Multi-Agent ─▶ RAG ─▶ AI SDLC ─▶ Production
```

### Modules Overview

| # | Module | Focus |
|---|--------|-------|
| 1 | [Introduction to Agentic AI](modules/Module_1_Introduction_to_Agentic_AI.md) | Core concepts, model foundations, AI APIs, and enterprise use cases |
| 2 | [Prompt Engineering and Context Design](modules/Module_2_Prompt_Engineering_and_Context_Design.md) | Prompting principles, structured agent prompts, tool use, context engineering |
| 3 | [MCP (Model Context Protocol)](modules/Module_3_MCP_Model_Context_Protocol.md) | MCP architecture, components, server development, enterprise design |
| 4 | [Building AI Agents](modules/Module_4_Building_AI_Agents.md) | Single-agent architectures, memory, autonomous execution, guardrails |
| 5 | [Multi-Agent Systems](modules/Module_5_Multi_Agent_Systems.md) | Multi-agent concepts, collaboration, frameworks, enterprise workflows |
| 6 | [RAG and Enterprise Knowledge Agents](modules/Module_6_RAG_and_Enterprise_Knowledge_Agents.md) | Enterprise RAG, agentic RAG, knowledge governance, knowledge agents |
| 7 | [AI-Augmented Software Engineering](modules/Module_7_AI_Augmented_Software_Engineering.md) | AI-assisted development, AI-augmented SDLC, secure dev, AI DevOps |
| 8 | [Production Agent Systems](modules/Module_8_Production_Agent_Systems.md) | Production architecture, observability, responsible AI ops, scaling |
